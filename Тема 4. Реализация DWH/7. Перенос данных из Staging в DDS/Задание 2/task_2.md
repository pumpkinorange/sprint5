Задание 2. Заполните таблицу dm_users
Сначала заполните таблицы, наиболее удалённые от центра «снежинки». Это необходимо для того, чтобы более близкие таблицы могли ссылаться на записи в более удалённых. Начните с измерения «Пользователи».
Создайте шаблон DAG и реализуйте логику заполнения dm_users. В качестве шаблона можете использовать DAG из уроков про заполнение stg-слоя.

подсказка:
В STG данные о пользователях лежат в формате JSON. Поэтому логика шага может выглядеть таким образом:
Прочитать данные из таблицы в STG: выгрузить в Python.
С помощью Python преобразовать JSON к объекту. Воспользуйтесь функцией str2json из файла lib/dict_util.py.
Вставить объекты в таблицу dds.dm_users.
